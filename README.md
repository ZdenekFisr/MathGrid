# MathGrid
A simple mathematical game, created using WPF (.NET Framework 4.7.2).
Your task is to fill all the empty fields in the grid with numbers to get the correct results in all rows and columns. You can use numeric pad to do so and also the arrows, keys "N" (new game), "R" (reset) and "F" (fix number; it makes your input resistant to resetting; also can be done by right-click). Difficulty (size of the grid) and ratio of visible numbers are selectable. Good luck!
